How it works: The refresh token is encrypted with RC4 using a key derived from your password + a random salt (10,000 key-stretching rounds). It is safe to hardcode in your app's source code — only your password can decrypt it. Refresh tokens don't expire unless revoked, solving the 4-hour token limit.
